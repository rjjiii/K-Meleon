Walk on Grass

Walk on Grass lets you run modern web engines in Windows XP. It's an extension for one of the last modern web browsers to support older PCs. K-Meleon is a lightweight, customizable web browser that still supports non-SSE processors, Windows XP, Windows Vista, and can run in limited RAM. K-Meleon receives weekly security patches and bug fixes. Unfortunately, new web tech is constantly being promoted by Google and some sites using these new technologies like web components may not display properly on K-Meleon. Walk on Grass adds a button to K-Meleon's toolbar and a right-click option for links, to open pages with a customized portable browser. Walk on Grass lets you use K-Meleon for most sites and then easily open less compatible pages in a new window.

Installation

To install Walk on Grass with the K-Meleon Extension Manager:

  1. Download the K-Meleon Extensions Manager for your operating system.
  2. Extract the extension manager packages directly into your K-Meleon folder.
  3. Go to Tools - Extension Manager.
  4. Click the Run Installer button. A file dialog will open. Browse to an extension saved in the .7z format.
  5. K-Meleon will automatically restart after an extension is installed.
Or to install Walk on Grass manually:
  1. Download the compressed file.
  2. Copy and Paste the root K-Meleon folder to create a backup.
  3. Extract the extension directly into your root K-Meleon folder.
  4. Restart and test K-Meleon.
