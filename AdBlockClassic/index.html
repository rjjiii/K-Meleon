AdBlock Classic for K-Meleon

AdBlock Classic is an open-source extension for K-Meleon. It is an updated and de-branded descendant of the classic AdBlock Plus extension for Firefox. It included a custom interface for K-Meleon. AdBlock Classic supports AdBlock Plus filters and rules (including AdBlock Locus's Fallback subscription). The package is open-source software available under the MPL 2.0. AdBlock Classic is distributed freely, as-is, and without warranty. That said, you're welcome to visit the K-Meleon forums if you have questions or issues.
