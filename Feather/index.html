Feather

Feather is a set of clean, elegant, modern icons available under the MIT license. This skin uses the Feather icons to build a complete 75-style theme (works with current releases of K-Meleon too, but not 74 for Windows 2000) with replacements for all icons at 16-48 pixel resolutions. The skin also includes some custom button templates, a Feather throbber, and hot/hover animations for buttons, even those that didn't have them in the default theme. The icons are mostly transparent and should work well with most Windows color schemes. You can use the solid color background.bmp file to set a custom background color.

Feather does not include any preferences or a custom toolbars configuration, so it will not change any settings or your current layout.

  1. Download the .zip file below.
  2. Extract the .zip file and place the Feather folder in your K-Meleon/skins/ folder or profile/skins/ folder.
  3. Open K-Meleon and press F2 to open the preferences window. Choose the Feather skin, set your toolbar size, and choose to enable to toolbar background (if you wish).
  4. Restart K-Meleon for changes to take affect.
