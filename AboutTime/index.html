About Time

About Time adds an About button to extensions in K-Meleon's Add-ons list. It's a build of Reality Ripple's extension "About Add-Ons" that will install to K-Meleon 75.1 without any configuration or modification. If you are running the current version of K-Meleon, you can use About Add-ons directly from Reality Ripple.
