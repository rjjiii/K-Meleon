Extension Hauler

Extension Hauler can export XUL-based extensions from K-Meleon. You can even export all of your extensions at once when you're upgrading to a new version of the browser. This is a build of Reality Ripple's extension "ExtExp" that will install to K-Meleon 75.1 without any special setup or modification. If you are running the current version of K-Meleon, you can simply use ExtExp directly from Reality Ripple.
