K-Meleon Tools



Extensions

AdBlock Classic

AdBlock Classic is an open-source ad-blocker for K-Meleon. It's an updated and de-branded descendant of the classic AdBlock Plus extension for Firefox. Ad Block Classic can use ABP filters and rules (including AdBlock Locus's Fallback subscription).


Password Extractor

Transfer passwords between browsers. This extension for K-Meleon can also be installed on other browsers that use XUL including SeaMonkey, Pale Moon, Mypal, and Waterfox Classic. Its XML export/import format is used by Password Exporter (for Firefox and SeaMonkey) and Password Backup Tool (for Pale Moon and Basilisk). It can also handle a simpler CSV export for popular browsers and password managers including Mozilla Firefox, Google Chrome, Microsoft Edge, Opera, Brave, Vivaldi, KeePass, and KeePassXC.


Walk on Grass

Walk on Grass adds a button to K-Meleon's toolbar and a right-click option for links, to open pages with a customized portable browser. There is an open-source Mozilla-derived browser for XP and a Chromium-derived browser. Walk on Grass lets you use K-Meleon for most sites and then easily open less compatible pages in a new window. Get the security of K-Meleon and the compatibility of Chromium.


AdBlock Locus

K-Meleon is a privacy-focused web browser that comes with a built-in adblocker (BAB). However, it does not come with a graphical interface for that adblocker. AdBlock Locus (ABL) provides a graphical interface to toggle the BAB, create custom rules, manage subscriptions, and create exceptions.


📤 Extension Hauler

Use Extension Hauler to export XUL-based extensions from K-Meleon. It makes upgrading versions a snap.


🛂 About Time

About Time adds an About button to extensions in K-Meleon's Add-ons list. You can pop up the about dialog for any extension from the "about:addons" page instead of combing through files.



Installers

Install K-Meleon for Windows 2000

There is an unstable port of K-Meleon 74 for Windows 2000 that includes the Goanna 2.2 engine, Transport Layer Security (TLS) support, and several fixes from recent Goanna versions. Most web browsers designed for Windows 2000 (and 98) did not support TLS and cannot access modern sites using the HTTPS protocol. This is the most modern web engine that can run on the Windows 2000 and Windows 9X operating systems.


Install K-Meleon 1.5 for Windows 98

This installer will install K-Meleon 1.5.4 (one of the last versions to support Windows 98) with Roytam's Transport Layer Security (TLS) patch. Most web browsers designed for Windows 9X systems did not support TLS and cannot access modern sites using the HTTPS protocol. The installer supports Windows 95, 98, 98 SE, Me, and 2000. Windows XP and later are supported in the current version of K-Meleon. You can visit the forum for more information about the TLS patch, or read the release notes for details about K-Meleon 1.5.4.



Reference Extensions

"Hello World" reference extensions.

Learn from these simple demo extensions. They all create "Hello world!" alerts but use several different programming techniques to demonstrate K-Meleon APIs like JSbridge and the macro language.
