AlphaAura (Legacy)

AlphaAura (Legacy) is a modified version of the Phoenity "AlphaAura" skin used by default in K-Meleon 74. The "alpha" in AlphaAura refers to how the original skin used alpha transparencies to smooth the icons. Those alpha transparencies are fully supported as of Windows XP but partial alpha channel transparencies are not available for the Windows 2000 interface. Version 74 was the last version of the K-Meleon shell to run on Windows 2000, so Roytam updated it to run with an older version of the Goanna engine and modern security ciphers to access modern web pages. To try out AlphaAura (Legacy):

  1. Download the .zip file below.
  2. Extract the .zip file and place the AlphaAura folder in your K-Meleon/skins/ folder or profile/skins/ folder. If you want to retain the original AlphaAura, you can rename this theme's folder to AlphaAura(Legacy).
  3. Open K-Meleon and press F2 to open the preferences window. Choose the AlphaAura or AlphaAuras (Legacy) skin and set your toolbar size to 16.
  4. Restart K-Meleon for changes to take affect.
